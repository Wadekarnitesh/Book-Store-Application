Java Web Application using Servlet, JSP, and JDBC (MySQL)

This project is a simple Java web application built using Servlets, JSP, and JDBC for database connectivity with MySQL.
It demonstrates the core concepts of Java EE web development, including request handling, MVC structure, form processing, database operations, and dynamic page rendering.
